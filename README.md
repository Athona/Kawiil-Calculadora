# Kawiil Calculadora  
Aplicación educativa de electrónica básica y aplicada

Kawiil Calculadora es una aplicación educativa desarrollada con **React** y **Capacitor**, orientada a facilitar el aprendizaje y la comprensión de conceptos fundamentales de **electrónica**, desde nivel **secundaria** hasta **nivel universitario inicial**.

El proyecto integra teoría, ejemplos prácticos y herramientas de cálculo, con el objetivo de servir tanto como apoyo académico para estudiantes como herramienta de consulta para técnicos e ingenieros.

---

## Objetivo del proyecto

El propósito de Kawiil Calculadora es proporcionar una plataforma educativa que:

- Apoye el aprendizaje progresivo de la electrónica.
- Refuerce la comprensión de conceptos fundamentales mediante la práctica.
- Ofrezca herramientas claras y confiables para el cálculo electrónico.
- Unifique explicación teórica y aplicación práctica en un solo entorno.

Adicionalmente, el proyecto funciona como un ejercicio de demostración de habilidades en desarrollo frontend, arquitectura de aplicaciones y diseño de software educativo.

---

## Enfoque educativo

Cada tema dentro de la aplicación se estructura bajo un enfoque pedagógico progresivo:

1. Introducción teórica del concepto  
2. Explicación detallada de las fórmulas involucradas  
3. Ejemplos resueltos paso a paso  
4. Calculadora interactiva  
5. Ejercicios o evaluaciones breves de refuerzo  

El objetivo no es únicamente obtener resultados numéricos, sino asegurar la comprensión del principio físico detrás de cada cálculo.

---

## Niveles de aprendizaje

La aplicación está diseñada para escalar por niveles académicos:

- **Secundaria:** conceptos básicos, lenguaje claro y ejemplos simples  
- **Bachillerato:** aplicación matemática y análisis intermedio  
- **Universidad (nivel introductorio):** fundamentos teóricos y casos prácticos  

El contenido se adapta según el nivel seleccionado por el usuario.
