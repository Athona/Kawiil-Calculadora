# Kawiil Calculadora  
Calculadora de electrónica básica

Kawiil Calculadora es una aplicación desarrollada con **React** y **Capacitor**, orientada al cálculo de valores comunes en **electrónica básica**. El proyecto está enfocado en ofrecer una herramienta clara, funcional y accesible para la resolución rápida de operaciones eléctricas.

Actualmente, la aplicación funciona como una calculadora especializada, con opciones para guardar y consultar resultados previos.

---

## Objetivo del proyecto

El objetivo principal de Kawiil Calculadora es proporcionar una herramienta práctica que permita:

- Realizar cálculos frecuentes en electrónica básica
- Guardar un historial de resultados
- Facilitar la consulta y reutilización de cálculos anteriores
- Ofrecer una interfaz clara y fácil de utilizar en dispositivos móviles

El proyecto también sirve como práctica y demostración de habilidades en desarrollo de aplicaciones con React y Capacitor.

---

## Funcionalidades actuales

- Calculadoras de electrónica básica  
- Almacenamiento de historial de cálculos  
- Visualización de resultados previos  
- Opción de impresión de resultados  
- Interfaz adaptable a distintos tamaños de pantalla  

---
